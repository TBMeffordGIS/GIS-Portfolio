# Interactive Utility Asset Viewer

## Project Summary

This project demonstrates how modern Web GIS can provide an intuitive interface for viewing and interacting with utility infrastructure assets.

Built with Leaflet.js, the application displays utility assets on an interactive web map. Users can click individual assets to view associated metadata, making spatial information easier to understand for engineers, GIS professionals, and field personnel.

Unlike traditional static maps, this project illustrates how browser-based GIS can support infrastructure management through interactive visualization.

---

## Objectives

- Display utility assets on an interactive web map
- Demonstrate Web GIS concepts using Leaflet
- Present asset metadata through map popups
- Create a responsive interface suitable for desktop and mobile devices
- Build a reusable foundation for future utility visualization projects
