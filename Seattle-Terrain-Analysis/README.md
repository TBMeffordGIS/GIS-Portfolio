# Seattle Terrain Analysis

Terrain analysis project created using QGIS and USGS 1 Meter DEM data.

## Outputs

- Terrain Visualization
- Hillshade Generation
- Contour Extraction
- DEM Mosaic
- Cartographic Layout

## Data Sources

- USGS 1 Meter DEM
- OpenStreetMap

## Software

- QGIS 3.44
- GDAL
